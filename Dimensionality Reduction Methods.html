Dimensionality Reduction Methods¶

Part I: Research Question¶

Can we predict identify the principal variables of the customers using PCA?

We will answer this question using principal component analysis (PCA).

The goal of this data analysis is to find if we can identify the principal variables of the customers.

Part II: Method Justification¶

PCAs aim is to reduce a larger set of variables into a smaller set of 'artificial' variables, called 'principal components', which account for most of the variance in the original variables. We would assume that PCA will find the most important components and the amount of components that matter the most, for example in this dataset, we would assume that out of all the variables we run PCA on, it will find out the most imprtant components in the set. PCA will analyze this dataset by splitting the continuous variables that we choose into smaller components. We will then analyze the variance of these components to find the components that matter the most to the overall analysis and dataset.

An assumption of PCA is that there needs to be a linear relationship between all variables.
